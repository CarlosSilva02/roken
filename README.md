# Roken

Roken is a web application that allows verified UTRGV users to safely buy and sell items within the platform.

## Features
- User authentication with email verification
- Secure user accounts
- Create, view, and manage listings
- In-app messaging between users
- Verified users only

## Technologies Used
- React
- Firebase Authentication
- Firestore Database
- JavaScript
- CSS

## Notes
- Users must verify their UTRGV email before accessing full features.
- Firebase is used as the backend API for authentication and data storage.
